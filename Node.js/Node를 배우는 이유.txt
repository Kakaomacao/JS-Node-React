JS는 브라우저 내장 자바스크립트 엔진을 이용하여 실행
ex) 사파리, 파이어폭스, 크롬, 엣지, 오페라
하지만 모두 다른 엔진

자바스크립트는 브라우저에서만 사용 가능
=> JS는 클릭, 경고창 띄우기 정도의 기능만 가능

크롬의 V8엔진은 C++로 개발되었기 때문에
JS를 브라우저가 아닌 곳에서도 실행 가능하게 해보자
=> Node 탄생 : 자바스크립트의 실행 환경, 런타임

독립적인 실행을 통해 자바스크립트의 태생적 한계가 해결
따라서 자바스크립트를 통해서 웹서버를 개발할 수 있음

React => 브라우저에서 동작하는 복잡하고 여러 기능을 하는 자바스크립트 파일들을 쉽게 생성하는 기술
웹 브라우저에 전달되어 프로그램과 같이 실행됨 + Node.js를 기반으로 사용할 수 있음
